# Demo Assets

This folder contains the public demo input and curated output images used in the repository README.

- `input/`
  Source image used for `img2img` examples.
- `output/`
  Curated demo outputs for `text2img`, `img2img`, and the final diary-page export shown in the README.
- `metadata/`
  Reproducibility notes for the demo assets such as prompt, preset, seed, and reference mode.
